# Teaching-System

教学系统的中枢仓库 — 文档、规范、技能。

## 目录结构

```
├── AGENTS.md              # 项目总文档（新 agent 必读）
├── skills/                # 项目专属技能（7个）
│   ├── README.md          # 技能索引 + 快速接管指南
│   ├── teaching_prep.md
│   ├── deploy_dashboard.md
│   ├── dashboard_ui.md
│   ├── teaching_system_maint.md
│   ├── system_audit.md
│   ├── batch_materials_ops.md
│   └── push_github.md
```

## 快速接管

1. 读 `AGENTS.md`（项目全貌）
2. 读 `skills/README.md`（技能索引）
3. 按需读具体技能文件
