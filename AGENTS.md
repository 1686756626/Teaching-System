## 二、备课流程（严格按顺序执行）

### 步骤 1：读取学生档案

从 `Student-Profiles` 仓库读取：
- `students.json` — 所有学生基本信息（姓名、年级、科目、上课时段）
- `profiles/{学生名}.md` — 详细档案（当前进度、薄弱点、上次反馈、学习风格）

> 如无法访问私有仓库，请要求教师手动提供学生信息。

### 步骤 2：读取教学日历

从 `Teaching-Calendar` 仓库读取：
- `schedule.json` — 结构化数据（优先使用）
- 或 `src/{学段}/{年级}/{科目}/{学期}.md`

确认学校当前周次和教学进度。

### 步骤 3：确定本周教学重点

对比学生进度 vs 学校进度：

| 情况 | 应对策略 |
|------|----------|
| 跟上 | 按学校进度走，巩固 + 适当抢进度 |
| 落后 | 优先补薄弱点，少量抢进度 |
| 超前 | 拓展拔高 + 真题训练 |

### 步骤 4：生成全部材料

生成以下文件，按规范写入 `Teaching-Materials/{学生名}/`：

```
{学生名}/
├── 学生/{周次}/
│   ├── {编号}-{科目}-{内容}（原卷）.md     ← 打印给学生
│   └── {编号}-{科目}-{内容}要点.md         ← 打印给学生
└── 教师/{周次}/
    ├── {编号}-{科目}-{内容}（解析）.md     ← 教师参考
    └── {编号}-{科目}-{内容}（教案）.md     ← 逐字稿
```

### 步骤 5：更新 README

更新 `Teaching-Materials/README.md`，同步目录结构和课表。

### 步骤 6：输出打印清单

给教师一个清晰的清单：
- 哪些文件需要打印（精确文件名）
- 课表概览（时间 - 科目 - 内容）

## 三、文件规范

### 3.0 文件命名规则

```
{编号}-{科目}-{内容}（类型）.md
```

| 元素 | 说明 | 示例 |
|------|------|------|
| 编号 | 两位数，按上课顺序 | 01、02、03 |
| 科目 | 语文 / 道法 / 历史 | |
| 内容 | 具体知识点 | 语法小测、社会规则要点 |
| 类型 | （原卷）/（解析）/（教案）/ 无后缀=要点 | |

编号规则：
- 编号 = 上课顺序，与时间轴对应
- 同一节课的原卷、解析、教案共享编号
- 学生用和教师用的编号互相对齐

### 3.1 原卷规范

- **绝对不能包含任何答案或提示**
- 顶部写明：姓名、日期、限时、满分
- 格式清晰，打印出来一目了然
- 禁止出现：参考答案、"打印时删除此部分"等提示

题目难度要求：
- **选择题**：不考单纯记忆，多考理解、辨析、综合
  - 材料型：给场景让学生判断
  - 比较型：对比两个概念
  - 组合型：多个要素混搭判断
  - 辨析型：表面合理实则有错
- **简答题**：每套至少2道，每道4分，贴合一模考法
  - 辨析题 / 情境分析题 / 比较题 / 史论结合题

### 3.2 解析规范

每道题必须包含：

| 要素 | 说明 |
|------|------|
| 标准答案 | 明确写答案 |
| 详细解析 | 为什么选这个，不选别的 |
| 对比表格 | 涉及混淆点时用表格对比 |
| 考点标注 | 标注考点名称和重要程度 |
| 做题技巧 | 总结通用解法 |
| 评分标准 | 简答题每点标明分值 |

- 与原卷题目一一对应
- 篇末附考点汇总表
- 语气面向教师

### 3.3 要点/框架类规范

- 多用表格、列表，少用大段文字
- 答题框架给**公式**，不给模糊思路
- 重要内容加粗，核心模板用引用块突出
- 末尾附速记口诀

### 3.4 教案（逐字稿）规范

这是最重要的输出——教师打开就能直接上课。

**必须包含：**

| 要素 | 格式 |
|------|------|
| 头部信息 | 学生姓名、日期、时长、学校当前进度 |
| 课前准备 | 打印清单（精确文件名）+ 课前要看的文件 |
| 要点文件对应 | 每个环节标注 `→ 对应 XX 要点文件位置` |
| 时间分配 | 精确到分钟（如 0-5min、35-40min） |
| 教师原话 | `你说："……"` |
| 板书内容 | `你在纸上写：……`（仅当要点文件中没有该内容时才写） |
| 过渡语 | 环节之间的衔接话术 |
| 分支判断 | 学生全对 / 错几道 / 错很多，分别怎么处理 |
| 课后布置 | 具体作业内容和要求 |
| 教学记录表 | 末尾附表格，课后填写 |

**要点文件对应规则：**
- 讲解时必须让学生翻开对应的要点文件
- 每个教学环节开头写：`→ 对应 XX 要点文件位置`
- 话术中包含"翻到要点第X部分""看要点上的XX表格"等指示
- 要点文件里已有的内容不重复板书

**语气：**
- 口语化大白话，不要教案腔
- ❌ "教师应引导学生思考"
- ✅ "涵松，来了？先别急，给你5分钟做一个小测"

**时间控制：**
- 总时长严格 ≤120 分钟
- 三科各约40分钟，预留5-10分钟弹性

## 四、内容生成原则

### 三科共用2小时的取舍

- 每科约40分钟，只聚焦 **1个核心知识点**
- 标准结构：小测5min → 核心15-20min → 休息5min → 抢进度10-15min → 练习+总结

### 知识点呈现

- 能用表格就不用文字
- 能给公式就不给思路
- 易混淆点用对比表格
- 简答题给"先写什么→再写什么→最后写什么"的框架

### 南昌市校历关键节点

| 月份 | 事件 |
|------|------|
| 9月 | 秋季开学 |
| 11月 | 期中考试（约第10-11周） |
| 1月 | 期末考试 + 寒假 |
| 2月 | 春季开学 |
| 4月 | 期中考试（约第10周） |
| 6月 | 高考(7-8日)、中考(中旬)、期末、暑假 |

### 特殊年级

- **九年级下学期**：冲刺节奏，非新课，重点复习+中考真题
- **高三下学期**：冲刺节奏，非新课，重点复习+高考真题
