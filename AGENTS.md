
## 四、备课流程（6 步）

### 步骤 1：读取学生档案

从 `Student-Profiles` 读取 `students.json` 和 `profiles/{学生名}.md`。
如无法访问私有仓库，请老师手动提供。

**班型与分组：**

| 班型 | 学生 | 材料 | 备注 |
|------|------|------|------|
| 一对二 | 刘晟廷 + 汪驿渃 | 共用一份，存刘晟廷目录 | — |
| 一对二 | 张翰中 + 曾梓瑞 | 共用一份，存张翰中，cp 到曾梓瑞 | — |
| 一对一 | 其余所有学生 | 各自独立 | — |

### 步骤 2：读取教学日历

从 `Teaching-Calendar/src/{学段}/{年级}/{科目}/{学期}.md` 读进度。
同步读取 `校历总览.md` 确认当前周次。

### 步骤 3：确定本周教学重点

对比学生进度 vs 学校进度：

| 情况 | 应对 |
|------|------|
| 跟上 | 按学校进度走，巩固 + 适当抢进度 |
| 落后 | 优先补薄弱点，少量抢进度 |
| 超前 | 拓展拔高 + 真题训练 |

> 📚 **查经验库：** 翻到下面「第六章：经验库」，看看有没有现成的教学模式和学生薄弱点解法可以直接用。

### 步骤 4：生成全部材料

**必须先读** `Teaching-Materials/备课规范.md`，了解格式要求。

每科每节课产出 **2 份文档**：

```
{学生名}/
├── 学生/{周次}/{编号}-{科目}-{内容}（学生版）.md     ← 学案（打印给学生）
└── 教师/{周次}/{编号}-{科目}-{内容}（教师版）.md     ← 时间轴操作手册
```

**学生版 = 学案：** 好的题目 + 要点总结，**不含答案**。
**教师版 = 操作手册：** 题目+答案+解析+逐字稿+分支判断全部融入时间轴。

> 📌 **标杆文件：** `杨紫泠/教师/W10/01-语文-期中备考冲刺（教师版）.md`

### 步骤 5：更新学生档案

记录课堂反馈、新发现薄弱点、遗留问题。

### 步骤 6：输出打印清单

精确文件名 + 课表概览。

## 五、文件命名规范

```
{编号}-{科目}-{内容}（学生版）.md
{编号}-{科目}-{内容}（教师版）.md
```

| 元素 | 说明 | 示例 |
|------|------|------|
| 编号 | 两位数，按上课顺序 | `01`、`02` |
| 科目 | 中文简称 | `语文`、`道法`、`历史` |
| 内容 | 本课主题 | `期中备考冲刺` |
| 类型 | `（学生版）`/ `（教师版）` | 括号包裹 |

**❌ 严禁写法：**

```
学生/黄涵松/W10/...       ← 学生名不能在第2层
黄涵松/学生/第10周/...     ← 周次不能加"第""周"
黄涵松/学生/W10/子目录/...  ← 不能增加第5层
```

## 六、📚 经验库

> **知识编译一次，持续更新，不从零推导。**
> 备课前翻一遍，避免踩重复的坑。

### 6.1 踩坑记录

#### 系统性坑

| # | 坑 | 后果 | 防范 |
|---|---|------|------|
| S1 | 校历年份没每学期核查 | 周次偏移2周 | 每学期开学跟老师确认开学日期 |
| S2 | parseSchoolCalendarMd 不 strip `**` | `items['开学']` 匹配不到 `**开学**` | 表格解析时 `.replace(/\*\*/g, '')` |
| S3 | loadCalendar() 异步不重渲染 | 首次打开显示旧周次 | `.then()` 里调 `this.route()` |
| S4 | 目录名格式不统一 | Dashboard 解析乱 | 统一用 `W{两位数字}` |
| S5 | Contents API 推中文文件 | 文件变 0 字节 | 走 git clone + push |

#### 备课内容坑

| # | 坑 | 正确做法 |
|---|---|---------|
| C1 | 作文练习只给标题 | 完整题面：导语+题目+要求 |
| C2 | 教师版只给解析不给原题 | 每个讲评 = 📝题目 + ✅答案与解析 |
| C3 | 作文用泛泛的"应急框架" | 速查表+速配训练（见6.2模式1） |
| C4 | 考前冲刺和常规课放反 | 考前冲刺在考试前一周，常规在前 |
| C5 | 一对二生成了两份不同材料 | 共用一份，cp 到副目录 |

### 6.2 验证有效的教学模式

#### 模式 1：初三一模考前作文冲刺

**适用：** 作文 38/50 左右，已学模板但不会套。**场景：** 一模/中考前最后一节课。

**核心：** 不教"怎么写好"，教"3分钟内定好素材和结构"。

```
0-15min    速查表讲解（题目大词→物件→故事→结尾）
15-35min   速配训练（3道真题，只练选素材）
35-50min   素材挖掘（每人挖2-3个真实素材）
50-55min   休息
55-110min  当场写一篇完整作文
110-120min 点评+修改
```

**速查表：** 题目大词 → 选具体物件（"封面折角的《活着》"而非"一本书"） → 真实故事 → 结尾升华

**参考：** `杨佐君浩/教师/W10/01-语文-一模考前作文冲刺（教师版）.md`

#### 模式 2：教师版时间轴

**核心原则：先练后讲**
- ✅ 小测→讲评→练习（先做）→讲评（答案+方法一起讲）→方法串讲
- ❌ ~~小测→讲评→知识讲解→练习→讲评~~

**必备元素：** 原题+答案+解析+"你说"逐字稿 | 分支判断预判具体错误 | 选择题逐项分析 | 标签 `⚠️薄弱点` / `⚡核心`

**标杆：** `杨紫泠/教师/W10/01-语文-期中备考冲刺（教师版）.md`

#### 模式 3：黄涵松类学生策略

**特征：** 只做舒适区的事，不擅长就逃避/撒谎/对抗

**策略：** 道法历史不硬背，用材料题带知识点 | 古诗文不信口头承诺，当场检查 | 作文核心问题是"不知道套哪个模板"和"细节写不具体"

### 6.3 学生薄弱点索引

> 按"问题"而非"学生"组织，快速定位解法。

**阅读理解答不精准：** 给答题框架（内容/氛围/情节/人物），不要靠感觉 → 杨紫泠、龙其乐、杨佐君浩、黄涵松

**阅读"变恶心了"：** 九年级难度陡增，答题没框架 → 龙其乐、杨佐君浩

**作文 38/50 上不去：** 速查表+速配训练，追问法挖具体物件 → 黄涵松、龙其乐、杨佐君浩

**古诗文背不下来还撒谎：** 当场默写，不信口头 → 黄涵松

**道法怕材料题：** 用材料题带知识点，不硬背 → 杨佐君浩、黄涵松

### 6.4 备课模板速查

**学生版骨架：**
```
# {科目}·{主题}  |  姓名：XXX  日期：____  限时：__分钟  满分：__分
一、小测（X分钟，X分）→ 二、练习（X分钟，X分）→ 三、要点总结
```

**教师版骨架：**
```
# {科目}·{主题}（教师版）|  学生：XXX  日期  时长  薄弱点
【课前准备】→【小测】→【小测讲评】→【练习】→【练习讲评 ⚡核心】→【方法串讲+课后安排】
```

**作业骨架（周中）：**
```
# {年级}语文作业·W{周次}周{X}  |  🎯知识点  ⏱约X分钟
4-5道题（基础→迁移→综合）→ ✅自查清单
```

**一对二同步检查：** 主副目录文件名一致 | 教师版写两人名字 | 学生版姓名栏写两人名字

### 6.5 每周 Lint 清单

每周一生成材料前跑一遍：

**自动检查：** 周次三处一致（校历/Dashboard/目录名）| 本周有课的学生材料不空 | 一对二材料 diff 一致 | 教师版每题有原题 | 作文有完整题面 | 学生版无答案

**人工检查：** 教学目标匹配学校进度 | 题目难度匹配学生水平 | 时间分配匹配 session 时长

## 八、周规划流程（每周一执行）

### 前置动作

1. 自动读取教学日历 → 列出本周学校进度
2. 自动读取学生档案 → 汇总薄弱点和遗留问题

### Checklist

```
□ 本周有没有考试/特殊事件？
□ 每个课段的科目和教学目标确认了吗？→ 先列方案，等老师确认
□ 上周遗留问题处理了吗？
□ 学生档案里有足够信息支撑备课吗？→ 不够就问
□ 材料生成优先级排好了吗？→ 按时间紧迫程度排序
```

### 关键原则

- **先出方案再干活** — 不要在老师确认前生成材料
- **信息不够就问** — 不要猜测学生情况
- **老师确认后才动手**
